# AI Symptom Triage Assistant

An AI-powered web application that analyzes user symptoms and provides:
- Possible conditions
- Urgency level
- Suggested next steps

## Tech Stack
- Backend: FastAPI
- Frontend: React
- AI: Groq (LLaMA 3)

## Features
- Real-time symptom analysis
- Emergency detection
- Structured AI responses

## Run Locally

### Backend
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
npm start
